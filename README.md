# vue-board-FE

Vue 3 + TypeScript + Vite 기반의 게시판 프론트엔드 프로젝트입니다.

## 기술 스택

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia (필요 시)
- Axios

## 개발 참고 문서

이 템플릿은 Vue 3 `<script setup>` SFC를 사용합니다.  
더 자세한 내용은 아래 문서를 참고하세요:

- https://v3.vuejs.org/api/sfc-script-setup.html
- https://vuejs.org/guide/typescript/overview.html#project-setup
